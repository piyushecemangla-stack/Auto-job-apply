üîß Browser Extension Setup
Note: This requires creating a simple Chrome/Firefox extension, but it's easier than terminal!

Create manifest.json

Create a new folder called jobbot-extension and create a file called manifest.json with:

{ "manifest_version": 3, "name": "JobBot AI", "version": "1.0", "description": "Auto-apply to jobs on Naukri and LinkedIn", "permissions": ["storage", "tabs", "activeTab"], "action": { "default_popup": "popup.html" }, "host_permissions": [ "https://www.naukri.com/*", "https://www.linkedin.com/*" ] }

Save this HTML as popup.html

Save this current page as popup.html in the same folder

Load Extension in Chrome

1. Open Chrome and go to chrome://extensions/

2. Enable "Developer mode" (top right)

3. Click "Load unpacked"

4. Select your jobbot-extension folder

5. Done! Click the extension icon to use it
